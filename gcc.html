安装 Windows 上的 GCC(MinGW)

列表

线程模型

posix/win32

喜欢哪个就选哪个(如果不出问题(笑))

异常处理模型

seh/sjlj (x64)

seh是新版
sjlj是旧版

dwarf/sjlj (x86)

dwarf是新版
sjlj是旧版

总结：sjlj 最稳定，兼容性最好。
对于新版，64位选 seh，32位选 dwarf。
 参考 "https://zhuanlan.zhihu.com/p/76613134"

安装 Linux 上的 GCC

sudo apt update
sudo apt-get install build-essential
